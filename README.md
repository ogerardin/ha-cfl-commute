# CFL Commute

Home Assistant integration for tracking Luxembourg CFL train commutes.

Fork of [adamf83/my-rail-commute](https://github.com/adamf83/my-rail-commute) (UK National Rail).

## Installation

### HACS
1. Open HACS → Explore → Add Repository
2. Search for "CFL Commute" or add: `https://github.com/ogerardin/hacs-cfl-commute`
3. Select "Integration"
4. Restart Home Assistant

### Manual
Copy `custom_components/cfl_commute` to `config/custom_components/`
